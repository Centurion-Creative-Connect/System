# サンプルシーンをインポートする

Centurion System のサンプルシーンは Unity Package Manager 上でインポートすることができます。

## Unity Package Manager を開く

上部メニューバーから `Window/Package Manager` をクリックで Package Manager を開きます。

## プロジェクト内の Package を見る

Package Manager ウィンドウ左上の `Packages: ...` をクリックし、`Packages: In Project` を選択します。

## Centurion System の Package を確認する

Package Manager ウィンドウ右側の `Packages - DerpyNewbie` の中にある `Centurion System` を選択します。

## Samples から Sample Scene をインポートする

Package Manager ウィンドウ左側の Description, Version History, Dependencies, Samples の中から、 Samples を選択し、
出てきた Sample Scene の Import ボタンをクリックします。

## インポートしたサンプルを確認する

`Assets/Samples/Centurion System/<version>/Sample Scene/` 内に 2 つのサンプルシーンが存在することを確認します。

- `CenturionSystemSampleScene.unity`
    - Player & Gun システムに加え、様々なギミックサンプルをまとめたシーンです。
- `CenturionSystemSampleSceneMinified.unity`
    - Player & Gun システムのみのシンプルなサンプルシーンです。
